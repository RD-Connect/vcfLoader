# vcfLoader

A tool to load and annotate vcf files to ElasticSearch through Spark and Hail (https://github.com/hail-is/hail).

The annotations used and their corresponding versions are:

* VEP (v88)
* dbNSP (2.9.3) 
* CADD (1.3)
* Clinvar (20180429)
* gnomAD exome (2.0.1)
* gnomAD genome (2.0.1)
* dbSNP (build 150)
* ExAC (0.3)
